Pengertian



Fungsi kubik adalah fungsi polinomial yang memiliki bentuk umum sebagai berikut:

f(x) = ax3 + bx2 + cx + d

Dimana :

  • a, b, c, dan d adalah konstanta.
  • a â‰  0 (agar fungsi tersebut memang fungsi kubik, jika a = 0 maka fungsi tersebut menjadi fungsi kuadrat).

Fungsi ini memiliki pangkat tertinggi 3, yaitu x3, yang membuat grafik fungsi ini memiliki bentuk khas, seperti melengkung dan bisa memiliki dua titik belok.

Sifat-sifat



Beberapa sifat utama dari fungsi kubik adalah :

Grafik

Grafik fungsi kubik selalu berupa kurva yang tidak simetris, dan dapat memiliki satu titik infleksi (titik belok), di mana grafiknya berubah dari cekung ke cembung atau sebaliknya.

Asimtot

Fungsi kubik tidak memiliki asimtot (garis yang didekati oleh grafik fungsi tetapi tidak pernah tercapai), karena fungsi ini berupa polinomial dan selalu terdefinisi untuk semua nilai.

Perilaku untuk nilai x yang lebih besar

  • Jika a > 0, grafik fungsi akan turun ke bawah di kiri dan naik ke atas di kanan.
  • Jika a < 0, grafik fungsi akan naik ke atas di kiri dan naik ke atas di kanan.

Titik stasioner

Fungsi kubik dapat memiliki hingga dua titik stasioner (titik di mana turunan pertama sama dengan nol), yang masing-masing dapat berupa titik maksimum atau minimum lokal.

Simetri

Fungsi kubik tidak memiliki simetri yang mudah, kecuali jika nilai-nilai tertentu dari koefisiennya membuat grafiknya lebih sederhana (misalnya, jika b = c = 0, fungsi menjadi fungsi f(x) = ax3 + d).

Contoh soal dan pembahasan



  • Contoh 1

    f(x) = 2x3 - 3x2 - 12x + 1

    Tentukan titik stasioner dan sifatnya (maksimum/minimum)

    Pembahasan :

    • Langkah pertama : Hitung turunan pertama f'(x)

      f'(x) = 6x2 - 6x - 12

    • Langkah kedua : Tentukan titik stasioner dengan menyamakan turunan pertama dengan nol

      6x2 - 6x - 12 = 0

      Bagi persamaan dengan 6

      x2 - x - 2 = 0

    • Langkah ketiga : Faktorkan persamaan kuadrat

      (x - 2)(x + 1) = 0

      Sehingga, x = 2 dan x = -1 adalah titik stasionernya.

    • Langkah keempat : Tentukan jenis titik stasioner dengan menghitung turunan kedua f"(x)

      f"(x) = 12x - 6

      Subtitusikan nilai x yang didapat dari langkah ketiga :

        untuk x = 2,

        f"(2) = 12(2) - 6 = 18

        untuk x = -1,

        f"(-1) = 12(-1) - 6 = -18

    • Kesimpulan :

      Titik stasioner x = 2 adalah titik minimum.

      Titik stasioner x = -1 adalah titik maksimum.

  • Contoh 2

    f(x) = x3 - 3x2 + 2

    Tentukan persamaan garis singgung pada titik x = 1

    Pembahasan :

    • Langkah pertama : Hitung turunan pertama f'(x)

      f'(x) = 3x2 - 6x

    • Langkah kedua : Hitung nilai f(1) dan f'(1)

      f(1) = 13 - 3(12) + 2 = 0

      f'(1) = 3(12) - 6(1) = -3

    • Langkah ketiga : Gunakan rumus persamaan garis singgung

      y - y1 = m(x - x1), dimana

      m = f'(1) dan

      (x1, y1) = (1, 0)

      Sehingga didapat,

      y - 0 = -3(x - 1)

      y = -3(x - 1)

      y = -3x + 3

    • Kesimpulan

      Persamaan garis singgung pada titik x = 1 adalah y = -3x + 3.

  • Contoh 3

    f(x) = 2x3 - 9x2 + 12x + 1

    Tentukan interval dimana fungsi tersebut naik atau turun!

    Pembahasan :

    • Langkah pertama : Hitung turunan pertama f'(x)

      f'(x) = 6x2 - 18x + 12

    • Langkah kedua : Faktorkan turunan pertama

      f'(x) = 6(x2 - 3x + 2)

      f'(x) = 6(x - 1)(x - 2)

      Sehingga didapat x = 1 dan x = 2

    • Langkah ketiga : Tentukan f'(x) pada inteval yang dibentuk oleh (-âˆž, 1), (1, 2), (2, âˆž)

      Untuk x < 1 (misal x = 0)

      f'(0) = 6(0 - 1)(0 - 2) = 12

      Karena hasilnya positif, maka dapat disimpulkan bahwa fungsi naik

      Untuk 1 < x < 2 (misal x = 1.5)

      f'(1.5) = 6(1.5 - 1)(1.5 - 2) = -6

      Karena hasilnya negatif, maka dapat disimpulkan bahwa fungsi turun

      Untuk x > 2 (misal x = 3)

      f'(3) = 6(3 - 1)(3 - 2) = 24

      Karena hasilnya positif, maka dapat disimpulkan bahwa fungsi naik

    • Kesimpulan

      Fungsi naik pada (-âˆž, 1) dan (2, âˆž)

      Fungsi turun pada (1, 2)
